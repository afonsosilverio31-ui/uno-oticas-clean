## Notes
- `mapUrl` in the database is assumed to be a valid Google Maps embed URL suitable for an `<iframe>` src attribute.
- WhatsApp numbers should be in international format without the '+' (e.g., 5511999999999) for wa.me links to work properly.
- Ensure `tailwind.config.ts` extends `fontFamily` with `display: ["var(--font-display)"]` to utilize the Outfit font.
